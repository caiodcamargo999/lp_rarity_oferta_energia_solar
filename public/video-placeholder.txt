O vídeo está hospedado no Cloudflare R2, não localmente.

Para otimizar o vídeo, você precisa:

1. CONFIGURAR VARIÁVEL DE AMBIENTE:
   Adicione no seu .env.local:
   NEXT_PUBLIC_VIDEO_URL=https://seu-bucket.r2.cloudflarestorage.com/video-de-vendas.mp4

2. OTIMIZAR O VÍDEO NO R2:
   - Faça download do vídeo do R2
   - Execute: npm run optimize-video
   - Faça upload do vídeo otimizado de volta para o R2
   - Atualize a URL no .env.local

3. O sistema irá usar:
   - Vídeo otimizado do R2 (principal)
   - API Proxy como fallback
   - Thumbnail atual como poster

Por enquanto, o sistema está usando o thumbnail atual como fallback.
